##Here is a sub-generator of [gulpfile-generator](https://github.com/bnjjj/gulpfile-generator)
#Usage
+ It's a submodules of [gulpfile-generator](https://github.com/bnjjj/gulpfile-generator) so if you want to try or change this sub-generator you have to download gulpfile-generator and develop into the submodule sass
+ It will create a sass task
+ For more information about configuration click [here](https://www.npmjs.com/package/gulp-sass)

#Tasks generated
+ gulp sass
+ gulp sass:watch

#Contributing
+ In the templates directory you'll see all of the task code about sass
+ Feel free to add more configurations in index.js in order to have more possibilities when creating this task

#Dependencies
+ [gulpfile-generator](https://github.com/bnjjj/gulpfile-generator)
